# Aplikasi BMI Java NetBeans

## Deskripsi
Aplikasi ini merupakan program sederhana berbasis Java GUI yang digunakan untuk menghitung Body Mass Index (BMI) berdasarkan berat badan dan tinggi badan pengguna.

## Fitur
- Input berat badan (kg)
- Input tinggi badan (cm)
- Perhitungan BMI otomatis
- Kategori hasil (Kurus, Normal, Gemuk, Obesitas)

## Teknologi
- Java
- Apache NetBeans IDE
- Java Swing (GUI)

## Cara Menjalankan
1. Buka project di Apache NetBeans
2. Klik Run
3. Masukkan berat dan tinggi badan
4. Klik tombol HITUNG
